# Tasks.org JSON Editor

A Visual Studio Code extension that provides a simple webview UI for opening, editing, and saving JSON files generated by the Tasks.org application.

## Features

* Open a `.json` file
* Display contents in an editable textarea
* Save modifications back to disk

## Usage

1. Run the **Open Tasks.org JSON Editor** command from the Command Palette.
2. Select a JSON file generated by Tasks.org.
3. Edit the raw JSON in the webview and click **Save**.
